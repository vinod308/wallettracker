This must be a valid Zoho Mail account
Displayed as sender name in client's inbox
Generate in Zoho → Account Security → App Passwords
All invoices CC'd here — configurable per send if needed
📋 HOW TO SET UP ZOHO APP PASSWORD
  1. Login to Zoho Mail → click your profile → My Account
  2. Go to Security → App Passwords
  3. Click "Generate New Password"
  4. Name it: GarageWalletTracker
  5. Copy the generated password — paste it in the field above
  6. Enable 2FA on the Zoho account first (required for App Passwords)
